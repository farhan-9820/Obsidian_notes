The **miner API** controls mining-related activities.

### Functions

- Start mining.
- Stop mining.
- Configure mining parameters.
- Set mining rewards address.

### Examples

- Enable mining on a private Ethereum network.
- Change the account receiving mining rewards.

### Typical Uses

- Private blockchain environments.
- Educational blockchain experiments.

### Note

On modern Ethereum, after the transition to [[Proof of Stake (blockchain) |Proof of stake]], traditional mining is no longer used on the main Ethereum network, making this API largely relevant for legacy or private-chain contexts.
