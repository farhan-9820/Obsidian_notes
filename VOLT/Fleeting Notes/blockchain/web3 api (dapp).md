### `web3` (Core Utilities)

This namespace provides global helper utilities for handling data types and retrieving basic client information.

- **What it does:** It returns the version of the node client running, and handles basic cryptographic conversions (like hashing strings via Keccak-256 or converting Eth units into Wei).

### Functions

- Data conversion.
- Cryptographic hashing.
- Unit conversion.
- Client information retrieval.

### Examples

- Convert Ether to Wei.
- Compute hashes.
- Retrieve node version information.

### Typical Uses

- Utility functions used throughout DApp development.
