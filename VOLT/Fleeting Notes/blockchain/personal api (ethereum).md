The **personal API** manages user accounts and private keys.

### Functions

- Create accounts.
- Unlock accounts.
- Lock accounts.
- Sign transactions.

### Examples

- Generate a new Ethereum account.
- Unlock an account before sending transactions.
- Digitally sign a transaction.

### Typical Uses

- Wallet management.
- Account administration.
- Transaction authorization.

### Security Concern

Exposing the personal API publicly can be dangerous because it handles sensitive account operations.
