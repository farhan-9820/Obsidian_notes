data types --> 
- structured = data in spreadsheets and relational database 
- unstructured  = txt, images, audio, social media post
- semi structured = json files , XML 

DBMS --> 
- relational = MySQL, PostgreSQL 
- Document = MongoDB 
- Key-value = Redis 

DBMS Language --> 
- Data Definition Language (DDL) = impacts the structure of the database *(Create a table; add/remove columns , change column names, data types or format)*
- Data Manipulation Language (DML) =  *Insert Into, Delete from, Update, Select:* manipulate/access the data values stored in the database.
